Pico-HSM / RSA-Cryptors 動作確認済みボード一覧

本ページは、Pico-HSM・RSA-Cryptors で動作確認済みの RP2040 搭載ボード一覧をまとめたものです。

■ 動作確認済みボード

1. Raspberry Pi Pico

  • 非Wi-Fiモデル
  • 専用ファームウェアファイル：pico-NoWIFI フォルダ内の Python ファイル
  • USB接続のみで利用可能

2. Raspberry Pi Pico W

  • Wi-Fi搭載モデル
  • 専用HSMファームウェアファイル：Pico-W フォルダ内の Python ファイル
  • USB接続 + Wi-Fi接続の両対応

3. RP-2040-Grove Board（互換ボード）

  • RP2040搭載の Grove 拡張ボード
  • 専用ファームウェアファイル：Customfirmware フォルダ内の Python ファイル
  • USB接続で動作確認済み
  • Wi-Fi機能は未搭載のため、Wi-Fi接続には非対応

■ ファームウェアに関する重要事項

  • 全てのボードは「専用ファームウェア」でのみ動作します。
  • 対応していないボードに書き込んだ場合：
  • ボードが破損することはありません
  • HSMファームウェアが起動せず機能しません
  • USBストレージとしては通常通りマウントされます
  • 正しいファームウェアを上書きすれば復旧できます

■ 未対応ボードについて

以下のような RP2040 ボードは、現在は動作保証外です：

  • 独自 USB VID/PID を持つサードパーティ製 RP2040 ボード
  • I2C/SPI/LED が異なる特殊基板
  • Wi-Fi モジュールが特殊仕様の Pico W 互換品
  • Wi-Fi モジュール非搭載ボードに Wi-Fi対応版を書き込んだ場合

※これらに専用ファームウェアを書き込んでも、起動しないだけで物理的な故障は発生しません。

■ 今後の対応予定

  • 追加互換ボードの調査中
  • ユーザー提供による動作報告の収集
  • 今後ファームウェア側で自動検出機能を強化予定
